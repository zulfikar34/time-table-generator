What is the different between let, const and var in javascript?

In javaScript variable is very common to store data. You can store data by using let, const and var . let and const are block scop while var is a global scop. you can re-declare and re-write with var but you can't with let and const.

What is the different between find(), filter() method?

Another inportant methods on Js are find() and filter(). Both work based on array. But there are some visible different appeared. Such as The find() method is used to get back the first element in an array that works as the given condition. while The filter() method is used to create a new array containing all elements that works as the given condition.

How do you select an element in the DOM using javaScript? Name a few methods?

DOM (Document Object Model) is also another important topic in Js. You can get data from user using this method. There are many ways to select element in the DOM. Such as :

1. document.getElementById()
2. document.getElementsByClassName()
3. document.getElementsByTagName()
4. document.querySelector()
5. document.querySelectorAll()
